\# Release Checklist



\## Pre-release



\- Update version

\- Update CHANGELOG

\- Update README

\- Run tests

\- Run lint

\- Run typecheck

\- npm pack --dry-run



\## Release



\- Create Git tag

\- Push tag

\- GitHub Release

\- Publish npm



\## Post-release



\- Verify npm package

\- Verify GitHub Pages

\- Verify CLI install
